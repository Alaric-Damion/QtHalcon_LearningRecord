# Halcon_LearningRecoed

#### 介绍
记载一些在编程及图像处理过程中的问题及解决方案，主要包括QT与C++的一些实用小项目，Halcon及Opencv的图像处理相关解决方案，主要是自己学习的一些记录内容，只是用于学习，涉及的相关借鉴内容的我尽量标明出处，如涉及版权问题，请联系删除。

#### 软件环境
Qt：5.9.6
vs2019
Halcon 12.0/17.0
