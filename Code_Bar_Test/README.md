# Code_Bar_Test

#### 介绍
测试扫码枪无焦点采集录入的程序


#### 开发环境

1. 编译器 MSVC2015 64bit
2. Qt 5.9.6


#### 使用说明

1. 实现无焦点获取扫码枪输入信息；
2. 判定与上次输入是否一致，重复扫码不录入；
3. 软件框架下可运行，全局状态下获取信息；
