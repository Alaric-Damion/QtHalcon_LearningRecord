# 1、文件介绍 #
> CameraDllTest文件夹    相机采图UI界面
- include   相机Dll包括头文件
- lib       相机Dll和lib
> Balser_GigeCamera文件夹  相机实现源码

# 2、功能 #
-打开相机、连拍、抓拍功能
-更改帧率、更改曝光、增益、一键曝光功能
